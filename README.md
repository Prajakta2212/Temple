Temple_App

A Flutter application for Shree Siddhivinayak Temple services, providing features like splash Screen , onboardingscreen , HomeScreen and user authentication using Firebase.

✨ Features
🔐 Firebase Authentication (Login / Signup)
🏠 Home screen with temple services
📱 Onboarding screen
💾 SharedPreferences for session management
🚪 Logout functionality
🎨 Beautiful UI inspired by temple theme


🛠️ Tech Stack
Flutter
Dart
Firebase Auth
Flutter Bloc (State Management)
Shared Preferences
Google Fonts

🚀 Getting Started
1️⃣ Clone the repository
git clone https://github.com/Prajakta2212/Temple.git
cd Temple_App
2️⃣ Install dependencies
flutter pub get
3️⃣ Run the app
flutter run
